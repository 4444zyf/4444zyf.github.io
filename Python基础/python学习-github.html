# python面向对象

  • 类(Class): 用来描述具有相同的属性和方法的对象的集合。它定义了该集合中每个对象所共有的属性和方法。对象是类的实例。
  • 类变量（属性）：类变量在整个实例化的对象中是公用的。类变量定义在类中且在函数体之外。类变量通常不作为实例变量使用。
  • 实例属性：在构造函数中定义，通过实例名访问，若通过类名访问会抛出异常。
  • 动态属性：类和实例都可以有动态属性。
  • 数据成员：类变量或者实例变量, 用于处理类及其实例对象的相关的数据。
  • self关键字：在一个类的函数里面，函数第一个参数是self，即实例本身，self相当于java的this关键字。
  • 继承：即一个派生类（derived class）继承基类（base class）的字段和方法。继承也允许把一个派生类的对象作为一个基类对象对待。例如，有这样一个设计：一个Dog类型的对象派生自Animal类，这是模拟"是一个（is-a）"关系（例图，Dog是一个Animal）。
  • 组合：一个类可以由其它类的概念构成。
  • 方法重写：如果从父类继承的方法不能满足子类的需求，可以对其进行改写，这个过程叫方法的覆盖（override），也称为方法的重写。

下面是一个例子：

from math import pi

class Circle:
    #这里可以有类属性，所有实例的类变量（属性）是同一个地址
    
    def __init__(self, radius):  # 构造函数，其中的属性是实例属性
        self.radius = radius
    def area(self):
        return pi * self.radius * self.radius
    def perimeter(self):
        return 2 * pi * self.radius
    
class Ring:
    def __init__(self, radius_outside, radius_inside):
        self.circle_inside = Circle(radius_inside)  # 组合的一个应用，运用了另一个类中的构造方法
        self.circle_outside = Circle(radius_outside)
        
    def area(self):
        return self.circle_outside.area() - self.circle_inside.area()
    def perimeter(self):
        return self.circle_inside.perimeter() + self.circle_outside.perimeter()
    
    
c1 = Circle(5)      # 调用的时候没有self参数，不需要new Circle来声明c1
c2 = Circle(4)
print(c1.area())
print(c1.perimeter())
print(c2.area())
print(c2.perimeter())

r1 = Ring(5, 4)
print(r1.area())
print(r1.perimeter())

类的继承：

class Parent:        # 定义父类
    parentAttr = 100
    def __init__(self):
        print "调用父类构造函数"
 
    def parentMethod(self):
        print '调用父类方法'
 
    def setAttr(self, attr):
        Parent.parentAttr = attr
 
    def getAttr(self):
        print "父类属性 :", Parent.parentAttr
 
class Child(Parent): # 定义子类
    def __init__(self):
        print "调用子类构造方法"
 
    def childMethod(self):
        print '调用子类方法'
 
c = Child()          # 实例化子类
c.childMethod()      # 调用子类的方法
c.parentMethod()     # 调用父类方法
c.setAttr(200)       # 再次调用父类的方法 - 设置属性值
c.getAttr()          # 再次调用父类的方法 - 获取属性值

# ufunc函数

ufunc函数全称为通用函数，是一种能对数组中的元素逐个进行操作的函数。ufunc针对数组进行操作，并且都以NumPy数组作为输出。使用ufunc函数比使用math库中的函数效率要高很多。

这里是一些常见的ufunc函数：

函数 使用方法

log()、logl()、log2() 对数函数
exp() 指数函数
cumsum()、cumproduct() 累计求和、求积

# ufunc函数的广播机制
a = np.arange(0, 20, 10).reshape(-1, 1) # 变形为1列的数组，行数自动计算
b = np.arange(0, 3)
print(a)
print(b)
print("a + b是：")
print(a + b)

下面是输出：

[[ 0]
 [10]]
[0 1 2]
a + b是：
[[ 0  1  2]
 [10 11 12]]

广播：是指不同形状的数组之间执行算数运算的方式。ufunc函数进行计算的前提是两个数组的维度相容。若两个数组的维度不同，NumPy会实行广播机制。广播机制有规则要求：参考 (opens new window)

  • 各输入数组的维度可以不相等，但必须确保从右到左的对应纬度值相等

  • 如果对应纬度值不相等，就必须保证其中一个为1

expr2 = y**2 + sin(y)*cos(y)+sin(z)
print("expr1=", expr1)
display(Latex(f"$$expr2={latex(expr2)}$$"))
print("y=5时，expr1=", expr1.subs(y, 5))                     # 带入一个值
print("y=2，z=3时，expr2=", expr2.subs({y:2,z:3}))           # 代入两个字母的值，并未计算
display(Latex(f"$$expr2={latex(expr2.subs({y:2,z:3}))}$$")) # 用LaTeX呈现上面的值
print("y=2，z=3时，expr2=", expr2.subs({y:2,z:3}).n())       # 代入两个字母的值，并进行计算

# 求微分方程（组）的符号解

SymPy库提供了dsolve函数求常微分方程的符号解。

在声明时可以使用Function()函数，如

>>> y = Funciton('y')
>>> y = symbols('y', cls = Function)   # 将符号变量声明为函数类型

求下列微分方程的通解：

(1) 齐次方程：y″−5y′+6y=0;

(2) 非齐次方程：y″−5y′+6y=xe2x.

from sympy import *
x = symbols('x')
y = symbols('y', cls = Function)
eq1 = diff(y(x), x, 2) - 5*diff(y(x), x) + 6*y(x)               # y要带着自变量走！！！
eq2 = diff(y(x), x, 2) - 5*diff(y(x), x) + 6*y(x) - x*exp(2*x)
print("齐次方程的解为：", dsolve(eq1, y(x)))
print("非齐次方程的解为：", dsolve(eq2, y(x)))

输出：

齐次方程的解为： Eq(y(x), (C1 + C2*exp(x))*exp(2*x))
非齐次方程的解为： Eq(y(x), (C1 + C2*exp(x) - x**2/2 - x)*exp(2*x))

求下列微分方程的解：

(1) 初值问题：y″−5y′+6y=0，y(0)=1，y′(0)=0；

(2) 边值问题：y″−5y′+6y=xe2x，y(0)=1，y(2)=0.

eq3 = y(x).diff(x, 2) - 5*y(x).diff(x) + 6*y(x)
eq4 = y(x).diff(x, 2) - 5*y(x).diff(x) + 6*y(x) - x*exp(2*x)

# dsolve的右括号记得要打在ics条件的后面
print("初值问题的解为：{}".format(dsolve(eq3, y(x), ics={y(0):1, y(x).diff(x).subs(x, 0):0})))
print("边值问题的解为：{}".format(dsolve(eq4, y(x), ics={y(0):1, y(2):0})))

输出：

初值问题的解为：Eq(y(x), (3 - 2*exp(x))*exp(2*x))
边值问题的解为：Eq(y(x), (-x**2/2 - x + 3*exp(x)/(-1 + exp(2)) + (-4 + exp(2))/(-1 + exp(2)))*exp(2*x))

# Pandas的序列和数据框

Pandas数据结构可以从一维到三维，Series(序列)是一维的，DataFrame(数据框)是二维的，Panel是三维甚至更高维的数据结构。

  • Series是个带标签的一维数组，可以存储任意类型的数据，例如，整型、浮点型、字符串和其他有效的Python对象。它的行标签称为index
  • DataFrame是个带标签的二维数组，由行和列，列可以由多种类型。DataFrame可以堪称二维结构的数组，例如电子表格和数据库表格。DataFrame也可以看作包含多个不同类型的Series的集合。
  • Pandas在统计学和经济学中，Panel data(面板数据)指多维数据，这个多维数据包括不同时间的不同测量结果、该数据结构的名称来源于其概念。Panel不常用。

# 文本文件的读取

read_csv调用有较多参数，其中比较重要的参数有：

pandas.read_csv参数详解 - 李旭sam - 博客园 (cnblogs.com) (opens new window)

pandas.read_csv — pandas 1.4.0 documentation (pydata.org) (opens new window)

  • filpath_or_buffer：可以是URL和文件，可用URL类型包括http、ftp等
  • sep：如果不指定参数，则会使用逗号分隔
  • delimiter：定界符、备选分隔符，如果选用该参数则sep失效
  • header：header=None，知名原始数据文件没有列标题，这样read_csv会自动加上列标题；header=0表示文件第一行为列标题（索引从0开始）；header参数可以是一个list，例如[1, 2, 3]，这个list表示将文件中的第1、2、4行作为列标题（意味着每一列有多个标题），这些行将被忽略掉
  • names：如果原数据集中没有字段，可以通过该参数在数据读取时给数据框添加具体的表头。
  • index_col：用作行索引的列编号或者列名，如果给定一个序列则有多个行索引
  • skiprows：数据读取时，指定需要调过原数据集开头的行数
  • nrows：指定读取数据的行数
  • na_values：指定原数据集中哪些特征的值作为缺失值。
  • skip_blank_lines：读取数据时是否需要跳过元数据及中的空白行，默认为True
  • parse_dates：如果参数值为True，则尝试解析数据框的行索引；如果参数为列表，则尝试解析对应的日期列；若果参数为嵌套列表，则将某些列合并为日期列；如果参数为字典，则解析对应的列（字典中的值），并生成新的字段名（字典中的键）
  • thousands：指定原始数据集中的千分位符
