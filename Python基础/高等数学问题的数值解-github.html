# 数值导数

利用泰勒级数可以给出近似计算函数导数的方法，例如，若f(x)有一阶导数，则泰勒级数：

(3.0)f(x+Δx)=f(x)+f′(x)Δx+o(Δx)

舍弃高阶无穷小并移项：

(3.1)f′(x)≈f(x+Δx)−f(x)Δx

精度一阶。由(3.1)还能推出二阶精度估计公式：

(3.2)f′(x)≈f(x+Δx)−f(x−Δx)2Δx

当函数具有更高阶的导数时，比如利用：

f(x+Δx)=f(x)+f′(x)Δx+f′′(x)2!(Δx)2+o((Δx)2)

以及

f(x−Δx)=f(x)−f′(x)Δx+f′′(x)2!(Δx)2+o((Δx)2)

可以得到二阶导数的计算公式：

(3.3)f′′(x)≈f(x+Δx)−2f(x)+f(x−Δx)(Δx)2

例 3.17 甲、乙、丙、丁4个人分别位于起始位置(-200, 200)、(200, 200)、(200, -200)以及(-200, -200)处（单位：m），并且以恒定的速率1m/s行走，在行走过程中，甲始终朝向乙的当前位置，同样，乙朝向丙、丙朝向丁、丁朝向甲。试绘制4人行走过程的近似路线。

分析：设位移为r(t)，则有速度v(t)=ddtr(t)，因此在很短的一段时间Δt内有近似：

r(t+Δt)≈r(t)+v(t)⋅Δt

位移、速度是矢量，因此在xOy平面内，又有

{rx(t+Δt)≈rx(t)+v(t)⋅Δt⋅cos⁡θ(t)ry(t+Δt)≈ry(t)+v(t)⋅Δt⋅sin⁡θ(t)

其中θ(t)是t时刻与x轴正向的夹角（这个式子对下面揭发其实用处不大）

# 数值积分

# 一重积分

在实际问题中，利用牛顿-莱布尼兹公式，通过求原函数，计算定积分：

∫abf(x)dx=F(x)|ab=F(b)−F(a)

梯形法的计算思想是把大的曲边梯形剖分成多个小的曲边梯形，每个小曲边梯形的面积用一个梯形面积做近似，最后累加求和得到所求定积分的数值解。

  1. 使用梯形公式时，把整个区间[a, b]进行n等分（此时步长h=(b−a)/n，分点为xi(i=0,1,⋯,n)，对应的函数值fi=f(xi)，这里xi=a+ih,i=0,1,⋯,n。积分∫abf(x)dx的数值解：

    (3.5)I1=h2(f0+2∑i=1n−1fi+fn)

    也可写成：

    (3.5)I1=h2(∑i=0nfi−(f0+fn))

  2. 使用辛普森公式时，将[a, b]进行2n等分，步长h=b−a2n，分点xi=a+ih，i=0,1,⋯,n。对应的函数值fi=f(xi)，积分∫abf(x)dx的数值解：

    (3.6)I2=h3[f0+2∑i=1n−1f2i+4∑i=1nf2i−1+f2n]

例 3.18 分别使用梯形公式、辛普森公式和SciPy工具库中函数quad求定积分

∫01sin⁡(cos⁡x+x2)dx
import numpy as np
from scipy.integrate import quad

def trapezoid(f, n, a, b):   # 定义梯形公式的函数
    h = (b - a)/n
    x_i = np.linspace(a, b, n + 1)
    return h*(np.sum(f(x_i))-(f(a) + f(b))/2)

def simpson(f, n, a, b):     # 定义辛普森公式
    h = (b - a)/(2*n)
    x_i = np.linspace(a, b, 2*n + 1)
    f_2i = f(x_i[::2])       # 数组索引，从第0个开始，步长为2
    f_2im1 = f(x_i[1::2])    # 从第1个开始，步长为2
    sum1 = np.sum(f_2i) - (f(a) + f(b))
    sum2 = np.sum(f_2im1)
    return h*(f(a) + f(b) + 2*sum1 + 4*sum2)/3

a = 0; b = 1; n = 1000
f = lambda x: np.sin(np.sqrt(np.cos(x) + x**2))
print("梯形积分I1: ", trapezoid(f, n, a, b))
print("辛普森积分I2：", simpson(f, n, a, b))
print("SciPy quad函数积分I3：", quad(f, a, b))

输出：

  • dblquad: dblquad(func, a, b, gfun, hfun, args=(), epsabs=1.49e-08, epsrel=1.49e-08)

    其中被积分函数func的格式为func(y, x)，最外层x的积分区间为[a, b]，内层y的积分区间为[gfunc(x), hfunc(x)]，返回元组，包括积分值和误差估计值

  • tblquad: tplquad(func, a, b, gfun, hfun, qfun, rfun, args=(), epsabs=1.49e-08, epsrel=1.49e-08)

    其中被积分函数func的格式为func(z, y, z)，最外层x的积分区间为[a, b]，中间层y的积分区间为[gfunc(X), hfunc(x)]，最内层z的积分区间为[qfun(x, y), rfunc(x, y)]

# 非线性方程（组）数值解

方程求解一直是数学中的核心问题之一，然而即使是对于形如：

∑i=0naixi=0

这样的代数方程， 当n≥5时也没有统一的求根公式。

常用的方程数值解法：二分法、牛顿迭代法

# 二分法求根

若f(x)∈C[a,b]（[a,b]上连续函数）且f(a)f(b)<0，则由介值定理，∃c∈[a,b]，使得f(c)=0，这时可以使用二分法对方程进行求根。

  1. 令a0=a,b0=b,n=0
  2. 令cn=(an+bn)/2
  3. 若|f(cn)|<ε，则算法停止，输出cn
  4. 若f(an)f(cn)<0，则an+1←an,bn+1←cn，否则，an+1←cn,bn+1←bn
  5. n←n+1，转至步骤2

采用二分法对方程求根时，第n次迭代对应的区间长度为(b−a)/2n，收敛速度是比较快的。

# 牛顿迭代法求根

牛顿迭代法_百度百科 (baidu.com) (opens new window)

若f(x)∈C2[a,b]（[a,b]上的二阶连续可微函数），f(a)f(b)<0且f′(x)在[a,b]上不变号，则方程f(x)=0在[a,b]内必然存在某个根x∗。设x0是x∗附近的点，则根据泰勒展开式有：

(3.7)0=f(x∗)=f(x0)+f′(x0)(x∗−x0)+f′′(ξ0)2(x∗−x0)2

令x1=x0−f(x0)f′(x0)，那么：

x∗−x1=x∗−x0+f(x0)f′(x0)=(3.7)−f(x0)−f′′(ξ0)2(x∗−x0)2f′(x0)+f(x0)f′(x0)=−f′′(ξ0)2f′(x0)(x∗−x0)2

即：

x∗−x1(x∗−x0)2=−f′′(ξ0)2f′(x0)

同样地，对每个i，若令：（这个式子是牛顿迭代公式）

(3.8)xi+1=xi−f(xi)f′(xi)

则有：

x∗−xi+1(x∗−xi)2=−f′′(ξi)2f′(xi)

若存在M=maxx∈[a,b]|f″(x)|minx∈[a,b]|f′(x)|，则：

|x∗−xi+1||x∗−xi|⩽M2|x∗−xi|

这说明该序列能够以较快速度收敛于x∗，该方法称为牛顿迭代法。

# 用SciPy工具库求解非线性方程（组）

例 3.21 求方程x3+1.1x2+0.9x−1.4=0实根的近似值，使得误差不超过10−6。用三种方法二分法、牛顿迭代法、直接调用SciPy求解。可以验证方程在(0,1)区间上有一个零点。

import numpy as np
from scipy.optimize import fsolve

def binary_search(f, eps, a, b):
    c = (a + b)/2
    while np.abs(f(c)) > eps:
        if f(a)*f(c) < 0: b = c
        else : a = c
        c = (a + b)/2
    return c

dx_donne = 1E-8           # 数值计算导数，设定dx大小
def diff(f, dx):
    return lambda x: (f(x + dx)-f(x - dx))/(2*dx) # 注意返回的是函数表达式
def newton_iter(f, eps, x0, dx):
    df = diff(f, dx)           # df变成了函数表达式
    x1 = x0 - f(x0)/df(x0)
    while np.abs(x1 - x0) >= eps:
        #x1, x0 = x1 - f(x1)/df(x1), x1，注意先计算等号右边，也就是说x1还没变成x0的值
        x0 = x1
        x1 = x0 - f(x0)/df(x0)
    return x1

f = lambda x: x**3 + 1.1*x**2 + 0.9*x - 1.4

result1 = binary_search(f, 1E-6, 0, 1)
result2 = newton_iter(f, 1E-6, 0, dx_donne)
result3 = fsolve(f, 0)

print("二分法得到的解：", result1)
print("牛顿迭代法得到的解：", result2)
print("用fsolve得到的解：", result3)

注意：python中多重赋值，变量赋值前，右边首先完成计算。右边的表达式从左到右计算

from scipy.optimize import minimize
f = lambda x: 100*(x[1] - x[0]**2)**2 + (1 - x[0])**2

print(x0)
print(x0.__class__)   # 注意x0是什么类！
